# PPhoenixLiveviewCourse

### Description

This is a course on Phoenix LiveView, a framework for building real-time web applications with Elixir and Phoenix.

This project and repository contains all the exercises and examples used in the course isolated by commits, it means each commit represents an exercise in the course.

### Installation

To start your Phoenix server:

- Run `mix setup` to install and setup dependencies
- Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
